วิธีเชื่อมต่อ Google Sheets
วิธีที่ 1 — Export CSV
กด Export CSV ใน "สรุปรายงาน" แล้ว Import เข้า Google Sheets
วิธีที่ 2 — Apps Script
ติดตั้ง Script แล้วกรอก URL เพื่อส่งข้อมูลอัตโนมัติ
ขั้นตอนติดตั้ง Apps Script
  1. เปิด Google Sheets → ส่วนขยาย → Apps Script
  2. ลบโค้ดเดิม แล้ว Copy โค้ดด้านล่างวางแทน
  3. กด บันทึก → Deploy → New deployment
  4. Type = Web app, Execute as = Me, Access = Anyone
  5. Copy URL แล้วนำมาวางในช่อง Webhook URL ด้านล่าง
